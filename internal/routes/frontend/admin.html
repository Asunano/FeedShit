API Token 管理

API Token 允许外部系统（CI、监控等）通过 POST /api/v1/external/feedback 提交反馈。请求头需携带 Authorization: Bearer <token>。

邮件通知模板

自定义邮件通知的标题和正文模板。支持占位符：{{project}} {{title}} {{description}} {{status}} {{admin_url}}

从 CSV 文件批量导入历史反馈数据。CSV 须包含 title 列，可选列：description, status, tags, assignee, contact_name, contact_email, priority, created_at（Unix 时间戳）。
